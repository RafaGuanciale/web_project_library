# Projeto 3: Biblioteca Triple Peaks

**vizualizar projeto:** https://rafaguanciale.github.io/web_project_library/

O projeto Biblioteca Triple Peaks foi desenvolvido como parte do Sprint 3 do curso de Desenvolvimento Web, onde o objetivo era:
- Criar uma página institucional moderna para uma biblioteca fictícia
- Focar em estrutura semântica, boas práticas de layout e organização do código
- Utilizar a metodologia BEM
- Boa utilização do posicionamento de elementos

## Funcionalidades

- Navegação intuitiva por meio de um menu fixo.
- Seção de Escolhas da Equipe, destacando livros recomendados.
- Sessão de Eventos, com destaque para palestras e atividades.
- Área de Associação, explicando o passo a passo para se tornar membro.
- Rodapé com informações de contato e redes sociais.

## Tecnologias utilizadas
- HTML5
- CSS3
- Normalize.css (para reset de estilos)
- Boas práticas de BEM na nomenclatura das classe

## Principais Funcionalidades e Soluções Técnicas

- Utilizar bem o posicionamento absoluto de elementos
- Uso funcional de flex-wrap para posicionar os cards dentro da seção

## Aprendizados Pessoais

- Estruturar páginas completas usando HTML semântico.
- Aplicar flexbox para posicionamento e responsividade.
- Utilizar bem hierarquia e consistência no CSS.
- Trabalhar com imagens, ícones e backgrounds de maneira eficiente.
- Uso adequado da posição absoluta de elementos
